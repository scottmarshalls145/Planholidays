Help & Documentation

Guides and troubleshooting

Quick Start Guide

1. Login: Use your username and password. Default admin is Plan / 2026plan.

2. Create Quotes: Click "Generate Quote" and fill in all 7 tabs.

3. Flights: Paste GDS text, upload images, or enter manually.

4. Hotels: Add hotels with images from Pexels or upload your own.

5. Payment: Enter total price and deposit. Instalments calculate automatically.

6. PDF: Click "Download PDF" for a professional A4 quotation.

7. History: All quotes are saved automatically. Search, edit, or export anytime.

Troubleshooting

Login not working? Check Caps Lock. After 3 failed attempts, account locks. Contact admin.

Data not saving? Check internet connection for Firebase sync. Local backup works offline.

PDF not generating? Ensure all required fields are filled. Try Preview first.

Images not showing? Pexels API may be rate-limited. Upload images manually instead.

Forgot password? Only administrators can reset passwords. Use the User Management screen.
